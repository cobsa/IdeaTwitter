# react_start_template
Startup code for react project. Configured to use babel and webpack with react

# Usage

Clone repo 
<code>
  git pull https://github.com/cobsa/react_start_template.git
</code>

<code>
  yarn
</code>
or 
<code>
npm install
</code>
and then run
<code>
 yarn run dev
</code>
or
<code>
  npm run dev
 </code>
